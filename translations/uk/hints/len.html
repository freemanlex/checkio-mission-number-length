Рядок у Python — це список символів. Отже, рядок «Hi» — це список із двох елементів «H» та «i». Тому майже всі функції cписку правильно працюють із рядком.

Тож твоя мета — перетворити ціле число на рядок, а потім знайти довжину цього рядка.

Яку функцію я шукаю?

Ось список усіх вбудованих функцій у Python. Спробуй подивитися на цей список, можливо щось потрібне знайдеш.

Чи можеш ти явно показати мені?
len() – це функція для визначення довжини, а str(), щоб перетворити об'єкт на рядок.
Добре, як мені ними скористатися?

Давай спочатку ти перетвориш дане ціле число у рядок. 

value = 56
text = str(value)
А як використати другу функцію?

Тепер знайди довжину text. 

value = 56
text = str(value)       
length = len(text)
То як має виглядати моє остаточне рішення?

Тобі лише потрібно обгорнути код у вираз функції і повернути length.

Ось повне рішення:

def number_length(value: int) -> int:

    text = str(value)
    length = len(text)

    return length

Або це можна записати в один рядок:

def number_length(value: int) -> int:

    return len(str(value))

Звісно, тримай!

Рекурсивно здійснюй цілочисельне ділення і рахуй рівень рекурсії поки value не стане 0. 

def number_length(value: int) -> int:

    return 1 + ((value := value // 10) and number_length(value))

Роби те ж, що і в попередньому розв'язку, але більш зрозуміло. Поки value має більше 1 цифри (> 9) додавай 1 до digits і роби цілочисельне ділення. Чому digits має початкове значення 1? Тому що невід'ємні числа менше 10 (навіть 0) мають довжину в 1 цифру. 

def number_length(value: int) -> int:

    digits = 1
    while value > 9:
        digits += 1
        value //= 10
        
    return digits
